1989 NeXT Dock

A faithful recreation of the 1989 NeXT workspace shelf aesthetic, optimized for modern web links.

This layout uses a fixed-width left sidebar (68px) with matte-black 48px square tiles. Each tile features a subtle 1px etched shadow effect characteristic of the NeXTSTEP UI.

Groups: Links are organized into logical groups separated by vertical hair-line rules (25% of the icon height).

Labels: Hover over any tile to reveal its identity in tiny, white, all-caps Chicago-style typography.
